Chrome Extension Installation

  1. Open Google Chrome
  2. Navigate to chrome://extensions/
  3. Enable "Developer mode" toggle in the top-right corner
  4. Click the "Load unpacked" button
  5. Select the dist-extension/ folder from this project
  6. The Leet2Git extension will appear in your extensions list
  7. Click the extension icon in your toolbar to open the popup
  8. Right-click the extension icon and select "Options" to configure settings
